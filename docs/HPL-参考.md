# HPL 语言参考

## 这是什么

HPL（Happy2018new's Python-AST Language）是社区开发者专为网易版 Minecraft 实现的自定义脚本语言，通过 Mod 插件内的解释器运行，非网易官方出品。

- **接口**：`{func, module.function, args...}` 是核心调用语法，一切外部交互——发指令、读选择器、操作数据、操控游戏内实体与方块——都通过它完成
- **指针**：复杂类型（`map`、`slice`、`tuple`、`set`）不能直接操作，只能通过整数句柄 + 内置函数间接访问
- **控制结构**：`if`/`elif`/`else`/`fi` 条件分支，`for`/`rof` 循环
- **表达式**：算术、逻辑、比较
- **变量与函数**：通过 `customfunction` 注册，支持参数和返回值

## 如何运作

HPL 是插件的**事件响应层**：所有 HPL 代码本质上都是被注册到某个触发源上的回调。触发源包括：

- **系统事件**：玩家进出、方块破坏、聊天等
- **UI 交互**：按钮点击、表单提交等

每条自定义指令由两部分组成：

- **触发源定义**：声明表单元素、绑定事件回调——决定"什么时候执行"
- **响应代码**：生成表单内容、处理提交数据——决定"执行时做什么"

## 如何导入

HPL 脚本通过指令导入服务器。每种指令声明一类代码单元——函数（`customfunction`）、表单（`customform`）、事件监听（`systemevent`）——由插件解释执行。
