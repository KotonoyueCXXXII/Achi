# 运算参考 — HPL 运算符速查

> 所有运算符、优先级、类型转换规则的紧凑参考。
> 表达式语法见 [expression.md](expression.md)，类型转换见 [statement.md](statement.md)。

## 逻辑运算

| 运算符 | 操作 | 示例 | 结果 |
|---|---|---|---|
| `and` | 与 | `True and False` | `False` |
| `or` | 或 | `True or False` | `True` |
| `not` | 取反 | `not True` | `False` |
| `in` | 成员检查 | `'ab' in 'abc'` | `True` |

**优先级（高→低）：** `in` → `not` → `and` → `or`

**短路求值：** `and` 遇假即停，`or` 遇真即停。非布尔值参与逻辑运算的结果不确定。

## 数学运算

| 运算符 | 操作 | 示例 | 结果 | 返回类型 |
|---|---|---|---|---|
| `+` | 加 | `1 + 2.0` | `3.0` | 任一操作数为 float 则返回 float，否则 int |
| `-` | 减 | `5 - 2` | `3` | 同上 |
| `*` | 乘 | `3 * 4` | `12` | 同上 |
| `/` | 除 | `7 / 2` | `3.5` | 始终返回 float |

优先级与标准四则运算一致。

## 比较运算

适用于数字和字符串，结果始终为 bool。**不可连用**——`1 < x < 10` 非法，须改写为 `1 < x and x < 10`。连用可能不报错但结果不确定。

| 运算符 | 操作 | 示例 | 结果 |
|---|---|---|---|
| `>` | 大于 | `3 > 2` | `True` |
| `<` | 小于 | `3 < 2` | `False` |
| `>=` | 大于等于 | `3 >= 3` | `True` |
| `<=` | 小于等于 | `2 <= 3` | `True` |
| `==` | 等于 | `1 == 1` | `True` |
| `!=` | 不等于 | `1 != 2` | `True` |

## 字符串运算

字符串比较按字典序逐字符比较：`'a' < 'b'`，`'ab' < 'b'`。所有比较运算符（`>`、`<`、`>=`、`<=`、`==`、`!=`）同样适用于字符串。

| 运算符 | 操作 | 示例 | 结果 |
|---|---|---|---|
| `+` | 拼接 | `'a' + 'b' + 'c'` | `'abc'` |
| `*` | 重复 | `'ak' * 5` | `'akakakakak'` |
| `==` | 相等 | `'aa' == 'ab'` | `False` |
| `!=` | 不等 | `'ac' != '666'` | `True` |
| `in` | 包含 | `'吃饭' in '我吃饭了'` | `True` |
