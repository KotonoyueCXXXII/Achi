# 数据类型参考 — HPL 支持的基本数据类型

> 所有类型及其字面量语法的权威定义。

## 整数 (int)

不带有小数点的数字。范围：−2147483648 ~ 2147483647（含）。超出范围的运算结果未定义。

## 布尔值 (bool)

`True` = 真/是，`False` = 假/否。

## 浮点数 (float)

带有小数点的数字。如 `2.3`、`5.20`、`1.0`。

## 字符串 (str)

用单引号 `'` 包裹的字符序列。如 `'你好'`、`'1+1=2'`。

**转义规则：** 字符串内使用反斜杠 `\` 转义特殊字符。目前仅支持对单个字符转义（不支持 `
` 形式的 Unicode 转义）。

| 转义符 | 用法 | 结果 |
|---|---|---|
| `\n` | 换行 | 换行符 |
| `\\` | 反斜杠 | `\` |
| `\'` | 单引号 | `'` |
| `\"` | 双引号 | `"` |

**命令方块注意事项：** 源代码在命令方块中用双引号 `"` 包裹，命令方块同样遵循上述转义规则。因此源码中的字符串需要额外转义：

```python
# 源代码（外层双引号已省略）
"'你好, 你知道 \\'Peter\\' 叫什么吗？'"

# 命令方块解析后（实际执行的代码）
'你好, 你知道 \'Peter\' 叫什么吗？'

# 最终字符串值
你好, 你知道 'Peter' 叫什么吗？
```

## 类型码速查（`ref_type` 返回值）

通过 `object.ref_type(ptr)` 可获取指针所指对象的类型码。`object.raw_type(val)` 仅支持基本类型（0–3），参数为原始值而非指针。

| `ref_type` | 类型 | 说明 |
|---|---|---|
| `0` | int | 基本类型 |
| `1` | bool | 基本类型 |
| `2` | float | 基本类型 |
| `3` | str | 基本类型 |
| `4` | None | key 不存在时 `ptr_get` 的返回值 |
| `5` | slice | 数组/切片 |
| `6` | map | 映射 |
| `7` | tuple | 元组 |
| `8` | set | 集合 |
| `0xFFFF` | unknown | 无效指针 |
